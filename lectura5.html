HTML 5 y CSS3

HTML es un lenguaje utilizado para crear y exponer el contenido en la web constantemente, a través de los diferentes navegadores. Este lenguaje ha ido evolucionando con el tiempo hasta llegar a lo que actualmente llamamos HTML5, Por otra parte, el CSS3 es un lenguaje de diseño de páginas web, compatible con todos los navegadores sin importar el dispositivo o sistema operativo.

Permite una división lógica entre la estructura de una página web (que maneja el HTML) y la forma en que la vemos. En definitiva, se trata del idioma utilizado para la aplicación de diseño de páginas web front-end.
